## 🔧 Quick Setup Guide - MSG91

### Step 1: Create Account
1. Visit: https://msg91.com
2. Click "Sign Up"
3. Enter:
   - Name
   - Email
   - Indian mobile number
   - Company name
4. Verify mobile via OTP
5. Verify email

### Step 2: Get Email API Credentials
1. Login to MSG91 dashboard
2. Go to "API" section
3. Create Email API key
4. Copy the Auth Key
5. Note your sender email

### Step 3: Add to Your App
```env
VITE_MSG91_AUTH_KEY=your-auth-key-here
VITE_MSG91_SENDER_EMAIL=noreply@yourdomain.com
```

### Step 4: Configure Sender
1. Add sender email in MSG91 dashboard
2. Verify sender email
3. Start sending!
