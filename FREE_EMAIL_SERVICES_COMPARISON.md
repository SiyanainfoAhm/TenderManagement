## 🚀 Quick Setup Guide (Brevo)

### Step 1: Create Account
1. Go to: https://www.brevo.com
2. Click "Sign up free"
3. Enter email and create account
4. Verify email

### Step 2: Get API Key
1. Go to Settings → SMTP & API
2. Create new API key
3. Copy the key
4. Add to `.env.local`:
   ```
   VITE_BREVO_API_KEY=your-api-key-here
   ```

### Step 3: Verify Sender
1. Add sender email (e.g., noreply@yourdomain.com)
2. Verify via DNS or single sender
3. Use this as "from" address

**Total Time:** 15 minutes  
**Cost:** $0 (Free forever for 300 emails/day)

---

## 🚀 Quick Setup Guide (Resend)

### Step 1: Create Account
1. Go to: https://resend.com
2. Sign up with GitHub or email
3. Verify account

### Step 2: Get API Key
1. Go to API Keys
2. Create new key
3. Copy the key
4. Add to `.env.local`:
   ```
   VITE_RESEND_API_KEY=your-api-key-here
   ```

### Step 3: Add Domain (Optional)
1. Add your domain
2. Or use resend.dev for testing
3. Configure DNS records

**Total Time:** 10-15 minutes  
**Cost:** $0 (Free for 100 emails/day)
